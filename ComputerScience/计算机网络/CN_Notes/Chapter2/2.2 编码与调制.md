### 2.2.1 编码与调制基本概念
- 编码：二进制数据 → 数字信号，逆过程称为“解码”  
- 调制：二进制数据 → 模拟信号，逆过程称为“解调”  
### 2.2.2 编码
![[../../CN_Picture/Pasted image 20251029165105.png]]
  
| 名称 | 规则 | 特点 |
|---|---|---|
| 不归零编码 (NRZ) | 低 0 高 1，中不变 | 无自同步能力，不浪费带宽，抗干扰弱 |
| 归零编码 (RZ) | 低 0 高 1，中归零 | 有自同步能力，浪费带宽，抗干扰弱 |
| 反向非归零编码 (NRZI) | 一跳 0 不跳 1 看起点，中不变 | 有自同步能力，不太浪费带宽，抗干扰弱 |
| 曼彻斯特编码 | 一跳 0 反跳 1 看中间，中必变 | 有自同步能力，浪费带宽，抗干扰强 |
| 差分曼彻斯特编码 | 一跳 0 不跳 1 看起点，中必变 | 有自同步能力，浪费带宽，抗干扰强 |
**归零**：可以作为**时钟**使用
注意：以太网默认使用曼彻斯特编码。
### 2.2.3 调制
**基带信号**：来自信源的数字信号，需**调制**后才能在某些信道上传输
- **调幅** (AM) / 幅移键控 (ASK)：
	- 若信号波有 $K$ 种振幅，则 1 码元 = $\log_2 K$ bit  
- **调频** (FM) / 频移键控 (FSK)：
	- 若信号波有 $K$ 种频率，则 1 码元 = $\log_2 K$ bit  
- **调相** (PM) / 相移键控 (PSK)：
	- 若信号波有 $K$ 种相位，则 1 码元 = $\log_2 K$ bit  
- **正交幅度调制** (QAM)：
	- AM 与 PM 结合，若有 $m$ 种振幅、$n$ 种相位，则 1 码元 = $\log_2 (m \cdot n)$ bit  
	- QAM-n的含义：采用 QAM 调制技术，有 n 种码元
