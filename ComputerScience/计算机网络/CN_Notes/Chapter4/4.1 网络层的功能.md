### 4.1.1 异构网络互联
- **异构** = 拓扑结构不同、物理层/链路层实现不同、主机类型各不相同。  
- **核心设备**：路由器（Router，TCP/IP 文献中又称 Gateway）。  
### 4.1.2 路由与转发
| 术语  | 定义与关系                                                     |
| --- | --------------------------------------------------------- |
| 路由  | 各个路由器根据路由协议生成路由表<br>相互配合规划 IP 数据报（分组）的最佳转发路径。             |
| 转发  | 根据转发表（路由表的精简版, 精简的数据结构有助于快速检索）<br>将收到的 IP 数据报从合适接口快速转发出去。 |
### 4.1.3 拥塞控制
#### 4.1.3.1 拥塞
- **拥塞原因**：网络出现过量分组 → 超负荷 → 性能下降
- **拥塞现象**：分组数增加，吞吐量反而降低  
#### 4.1.3.2 拥塞控制方法
- **开环控制（静态）**：网络部署前预先设计，运行后不再修改。  
- **闭环控制（动态）**：实时监视网络状态，发现拥塞后通过 ICMP 等传递拥塞信息，相关路由器及时调整路由表。
